# Video Script: CAPA - Unit 5.1
## Evaluating CAPA Effectiveness
**Duration:** 18 minutes

---

### Introduction (0:00 - 1:00)
**Speaker:** In this lecture, we will discuss evaluating CAPA effectiveness. Evaluating effectiveness is essential to ensure that the CAPA system is working correctly.

### Importance of Evaluating Effectiveness (1:00 - 5:00)
**Speaker:** Evaluating CAPA effectiveness is essential for several vital reasons.

**Main Reasons:**
1. **Quality Assurance:** Ensure that CAPA system is effective
2. **Performance Improvement:** Improve CAPA system performance
3. **Compliance:** Regulatory compliance
4. **Learning:** Learning from experience
5. **Continuous Improvement:** Continuous system improvement

**Detailed Explanation:**
- Evaluation must be periodic
- Evaluation must be comprehensive
- Evaluation must be documented
- Evaluation must be used for improvement

### Effectiveness Evaluation Criteria (5:00 - 10:00)
**Speaker:** There are several criteria to evaluate CAPA effectiveness.

**Main Criteria:**

**1. Problem Solving:**
- Are problems solved effectively?
- Are problems solved on time?
- Are problems solved permanently?
- Are problems solved without negative effects?

**2. Preventing Recurrence:**
- Did it prevent problem recurrence?
- Did problem frequency decrease?
- Did performance improve?
- Did quality improve?

**3. Efficiency:**
- Is the system efficient?
- Is the system cost-effective?
- Is the system time-effective?
- Is the system resource-effective?

**4. Compliance:**
- Is the system compliant with regulations?
- Is the system reviewed regularly?
- Is the system documented correctly?
- Is the system audited regularly?

### Effectiveness Evaluation Procedures (10:00 - 14:00)
**Speaker:** Evaluating effectiveness requires following specific procedures.

**Procedures:**

**1. Define Evaluation Criteria:**
- Define acceptance criteria
- Define performance metrics
- Define timeline
- Define evaluation methods

**2. Collect Data:**
- Collect performance data
- Collect quality data
- Collect cost data
- Collect time data

**3. Analyze Data:**
- Analyze collected data
- Compare data with criteria
- Determine effectiveness
- Identify improvement areas

**4. Evaluate Results:**
- Evaluate system effectiveness
- Evaluate system efficiency
- Evaluate compliance
- Evaluate performance

**5. Develop Improvements:**
- Develop system improvements
- Develop process improvements
- Develop procedure improvements
- Develop resource improvements

### Effectiveness Evaluation Tools (14:00 - 18:00)
**Speaker:** There are several tools that can be used to evaluate CAPA effectiveness.

**Main Tools:**

**1. Dashboards:**
- Display performance data
- Display quality data
- Display cost data
- Display time data

**2. Periodic Reports:**
- Monthly reports
- Quarterly reports
- Annual reports
- Special reports

**3. Internal Audit:**
- Periodic system audit
- Comprehensive system audit
- Process audit
- Procedure audit

**4. Management Review:**
- Periodic management review
- Comprehensive management review
- Performance review
- Improvement review

### Summary (18:00)
**Speaker:** In this lecture, we learned:
- Importance of evaluating effectiveness
- Effectiveness evaluation criteria
- Effectiveness evaluation procedures
- Effectiveness evaluation tools

**Key Points to Remember:**
- Evaluating effectiveness is essential
- Criteria must be clear
- Procedures must be followed
- Tools help in evaluation
