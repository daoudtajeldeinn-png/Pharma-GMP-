# Video Script: Validation - Unit 3.1
## IQ (Installation Qualification)
**Duration:** 18 minutes

---

### Introduction (0:00 - 1:00)
**Speaker:** In this lecture, we will discuss IQ or Installation Qualification. IQ is the first stage of validation and is essential to ensure correct equipment installation.

### What is IQ? (1:00 - 5:00)
**Speaker:** IQ is the verification of equipment installation.

**Definition:**
- Installation qualification
- Ensure installation follows specifications
- Ensure installation meets requirements
- Ensure installation is safe

**Importance of IQ:**
- Ensures correct installation
- Reduces future problems
- Ensures employee safety
- Essential for subsequent validation stages

**Detailed Explanation:**
- IQ must be performed immediately after installation
- IQ must be fully documented
- IQ is essential before OQ and PQ

### IQ Contents (5:00 - 12:00)
**Speaker:** IQ includes several elements that must be verified.

**Main Elements:**

**1. Site Verification:**
- Ensure site is appropriate
- Ensure site meets requirements
- Ensure site is safe
- Ensure site facilitates maintenance

**2. Facilities Verification:**
- Verify electricity availability
- Verify water availability
- Verify steam availability
- Verify compressed gas availability

**3. Equipment Verification:**
- Verify equipment matches specifications
- Verify equipment is intact
- Verify equipment is complete
- Verify equipment operates

**4. Installation Verification:**
- Verify correct installation
- Verify connections
- Verify mounting
- Verify ventilation

**5. Documentation Verification:**
- Verify document availability
- Verify document completeness
- Verify document accuracy
- Verify document approval

### IQ Procedures (12:00 - 16:00)
**Speaker:** Executing IQ requires following specific procedures.

**Steps:**

**1. Review Specifications:**
- Review technical specifications
- Review drawings
- Review installation guidelines
- Review safety requirements

**2. Site Inspection:**
- Inspect designated site
- Inspect available facilities
- Inspect surrounding environment
- Inspect accessibility

**3. Equipment Inspection:**
- Inspect equipment upon receipt
- Inspect equipment for integrity
- Inspect equipment for completeness
- Inspect equipment for compliance

**4. Installation Monitoring:**
- Monitor installation process
- Ensure installation guidelines are followed
- Ensure work quality
- Address any problems

**5. Installation Testing:**
- Test electrical connections
- Test mechanical connections
- Test safety systems
- Test basic functions

**6. Documentation:**
- Document all activities
- Prepare IQ report
- Review report
- Approve report
