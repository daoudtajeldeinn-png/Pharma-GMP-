# Video Script: GMP Basics - Unit 4.2
## Document Management
**Duration:** 20 minutes

---

### Introduction (0:00 - 1:00)
**Speaker:** In this lecture, we will discuss document management. Good document management is essential to ensure documents are up-to-date, accessible, and traceable.

### Document Management Principles (1:00 - 5:00)
**Speaker:** Document management encompasses all aspects of the document lifecycle from creation to disposal.

**Basic Principles:**
1. **Document Control:** Ensuring documents are current and compliant
2. **Traceability:** Tracking all changes
3. **Access:** Ensuring easy access to documents
4. **Security:** Protecting documents from damage or tampering
5. **Retention:** Retaining documents for the required period

**Detailed Explanation:**
- Every document must have a unique number
- Every document must have a version
- Every change must be documented
- Old documents must be archived

### Document Lifecycle (5:00 - 10:00)
**Speaker:** The document lifecycle includes several stages from creation to disposal.

**Stages:**

**1. Creation:**
- Drafting the document
- Content review
- Initial approval

**2. Review and Approval:**
- Review by experts
- Review by quality
- Final approval

**3. Issuance:**
- Distributing the document
- Training employees
- Implementing the document

**4. Use:**
- Using the document in daily work
- Monitoring effectiveness
- Collecting feedback

**5. Update:**
- Periodic review
- Content update
- Re-approval

**6. Disposal:**
- Archiving the document
- Retaining for required period
- Safe disposal

### Change Control (10:00 - 14:00)
**Speaker:** Change control is essential to ensure documents are current and compliant.

**Change Control Procedures:**
1. **Change Request:** Requesting document modification
2. **Change Evaluation:** Evaluating change impact
3. **Change Approval:** Approving the modification
4. **Change Implementation:** Implementing the modification
5. **Communication:** Informing employees of the change
6. **Training:** Training employees on the change

**Detailed Explanation:**

**Change Request:**
- Change request form
- Reason for change
- Proposed changes

**Change Evaluation:**
- Evaluating impact on quality
- Evaluating impact on compliance
- Evaluating impact on operations

**Change Approval:**
- Approval from quality
- Approval from management
- Approval from experts

**Change Implementation:**
- Updating the document
- Reviewing approval
- Issuing updated document

**Communication:**
- Informing all affected employees
- Explaining changes
- Explaining effective date

**Training:**
- Training on changes
- Evaluating understanding
- Documentation

### Document Numbering System (14:00 - 17:00)
**Speaker:** A document numbering system is essential to track and organize documents.

**Document Number Elements:**
1. **Department Code:** Identifies the department or function
2. **Document Type:** Identifies document type (SOP, FORM, etc.)
3. **Serial Number:** Unique number for the document
4. **Version Number:** Identifies document version

**Example:**
- SOP-QC-001-01
- SOP = Standard Operating Procedure
- QC = Quality Control department
- 001 = Serial number
- 01 = First version

**Detailed Explanation:**
- Each department has a unique code
- Each document type has a unique code
- Serial numbers are assigned sequentially
- Version increases with each update

### Document Retention (17:00 - 20:00)
**Speaker:** Document retention is essential for review and regulatory compliance.

**Retention Periods:**
- **Active Documents:** Throughout usage period
- **Obsolete Documents:** 1-5 years after discontinuation
- **Records:** 5-25 years depending on type
- **Regulatory Documents:** Up to 30 years

**Retention Procedures:**
- Archive centrally
- Easy access
- Protection from damage
- Good indexing

**Detailed Explanation:**
- Batch records: 1-5 years
- Training records: Employment duration + 5 years
- Maintenance records: Equipment lifetime + 5 years
- Inspection records: 5-10 years

### Summary (20:00)
**Speaker:** In this lecture, we learned:
- Document management principles
- Document lifecycle
- Change control
- Document numbering system
- Document retention

**Key Points to Remember:**
- Document management is essential for GMP
- Document lifecycle includes several stages
- Change control is essential
- Numbering system is essential for tracking
- Retention is essential for review
