# Video Script: GMP Basics - Unit 4.3
## Batch Records (BMR/BPR)
**Duration:** 22 minutes

---

### Introduction (0:00 - 1:00)
**Speaker:** In this lecture, we will discuss batch records (BMR/BPR). Batch records are essential to track the production of each batch and ensure its quality.

### What Are Batch Records? (1:00 - 5:00)
**Speaker:** Batch records are documents used to record all details of producing a specific batch.

**Definition:**
- BMR = Batch Manufacturing Record
- BPR = Batch Production Record
- Both refer to the same document

**Purpose:**
- Track production of each batch
- Ensure all processes are executed correctly
- Provide proof of quality
- Provide a record for traceability

**Basic Principles:**
- Each batch has its own record
- Record must be filled during production
- Record must be signed by responsible personnel
- Record must be kept for review

### Batch Record Contents (5:00 - 12:00)
**Speaker:** The batch record contains comprehensive information about the batch.

**Main Sections:**

**1. Batch Information:**
- Batch number
- Product name
- Batch size
- Production date
- Expiration date

**2. Raw Material Information:**
- Raw material numbers used
- Raw material quantities
- Raw material batch numbers
- Raw material receipt dates

**3. Equipment Used:**
- Equipment numbers
- Equipment status
- Maintenance schedules
- Calibration schedules

**4. Operating Procedures:**
- Procedures followed
- SOP numbers
- Any deviations from procedures

**5. Process Parameters:**
- Temperature
- Pressure
- Time
- Speed
- Any other parameters

**6. In-Process Quality Tests:**
- Test results
- Test times
- Any deviations
- Corrective actions

**7. Personnel:**
- Employee names
- Employee signatures
- Attendance times
- Responsibilities

**8. Documentation:**
- All observations
- Any problems
- Any corrective actions
- Final signature

### Preparing Batch Record (12:00 - 16:00)
**Speaker:** Preparing the batch record must be done correctly to ensure information accuracy.

**Steps:**

**1. Review Formula:**
- Ensure formula is current
- Ensure formula is approved
- Ensure formula is suitable for the product

**2. Prepare Form:**
- Use approved form
- Include all required sections
- Ensure form clarity

**3. Review Raw Materials:**
- Ensure raw material availability
- Ensure raw material quality
- Ensure raw material validity

**4. Review Equipment:**
- Ensure equipment readiness
- Ensure equipment maintenance
- Ensure equipment calibration

**5. Review Personnel:**
- Ensure personnel availability
- Ensure personnel training
- Ensure personnel competency

### Using Batch Record (16:00 - 20:00)
**Speaker:** Using the batch record during production is essential to ensure recording accuracy.

**Procedures:**

**Before Production:**
- Review batch record
- Ensure information completeness
- Sign to start

**During Production:**
- Record all parameters
- Record all tests
- Record any deviations
- Sign each step

**After Production:**
- Review record
- Ensure completeness
- Final signature
- Archive record

**Detailed Explanation:**
- Recording must be in real-time
- Recording must be accurate
- Recording must be legible
- Recording must be indelible

### Reviewing Batch Record (20:00 - 22:00)
**Speaker:** Reviewing the batch record is essential to ensure information accuracy and compliance.

**Reviews:**
1. **Self-Review:** By employees
2. **Supervisor Review:** By direct supervisor
3. **Quality Review:** By quality department
4. **Final Review:** By management

**What is Reviewed:**
- Completeness of all sections
- Information accuracy
- Signatures
- Any deviations
- Corrective actions

**Detailed Explanation:**
- Review must be comprehensive
- Any errors must be corrected
- Corrections must be documented
- Record must be approved before release

### Summary (22:00)
**Speaker:** In this lecture, we learned:
- What are batch records
- Batch record contents
- Preparing batch record
- Using batch record
- Reviewing batch record

**Key Points to Remember:**
- Batch records are essential for tracking production
- Record contains comprehensive information
- Record must be filled during production
- Review is essential for release
