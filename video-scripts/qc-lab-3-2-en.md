# Video Script: QC Lab & ISO/IEC 17025 - Unit 3.2
## Participation in External Testing
**Duration:** 18 minutes

---

### Introduction (0:00 - 1:00)
**Speaker:** In this lecture, we will discuss participation in external testing. Participation in external testing is essential to ensure laboratory competence.

### What are External Tests? (1:00 - 5:00)
**Speaker:** External tests are tests where laboratory results are compared with results from other laboratories.

**Definition:**
- Comparison tests between laboratories
- Proficiency tests
- Calibration tests
- Comparison tests

**Importance of External Tests:**
- Ensures laboratory competence
- Ensures result accuracy
- Ensures result reliability
- Essential for accreditation

**Detailed Explanation:**
- External tests must be periodic
- External tests must be comprehensive
- External tests must be documented
- External tests must be followed up

### Types of External Tests (5:00 - 10:00)
**Speaker:** There are several types of external tests.

**Main Types:**

**1. Proficiency Testing:**
- Tests to evaluate laboratory competence
- Tests to evaluate result accuracy
- Tests to evaluate result reliability
- Tests to evaluate laboratory performance

**2. Interlaboratory Comparisons:**
- Compare results between laboratories
- Compare analytical methods
- Compare equipment
- Compare personnel

**3. Calibration Testing:**
- Tests to calibrate equipment
- Tests to calibrate standards
- Tests to calibrate methods
- Tests to calibrate procedures

**4. Reference Testing:**
- Tests using reference standards
- Tests using reference materials
- Tests using reference methods
- Tests using reference laboratories

### Procedures for Participation in External Testing (10:00 - 14:00)
**Speaker:** Participation in external testing requires following specific procedures.

**Procedures:**

**1. Registration:**
- Register in programs
- Select appropriate tests
- Pay fees
- Obtain samples

**2. Analysis:**
- Analyze samples according to method
- Analyze samples within specified time
- Analyze samples accurately
- Analyze samples comprehensively

**3. Reporting:**
- Report results to organizer
- Report results within specified time
- Report results accurately
- Report results completely

**4. Evaluation:**
- Evaluate results
- Compare results with standards
- Identify deviations
- Identify improvement areas

### Evaluating External Test Results (14:00 - 18:00)
**Speaker:** Evaluating external test results is essential to ensure effectiveness.

**Procedures:**

**1. Review Results:**
- Review results from organizer
- Review performance evaluation
- Review deviations
- Review recommendations

**2. Analyze Performance:**
- Analyze overall performance
- Analyze performance for each test
- Analyze performance for each method
- Analyze performance for each employee

**3. Identify Improvement Areas:**
- Identify improvement areas
- Determine priorities
- Define required resources
- Define timeline

**4. Develop Corrective Actions:**
- Develop corrective actions
- Develop preventive actions
- Develop improvements
- Execute actions

### Summary (18:00)
**Speaker:** In this lecture, we learned:
- What are external tests
- Types of external tests
- Procedures for participation in external testing
- Evaluating external test results

**Key Points to Remember:**
- External tests are essential
- Participation must be periodic
- Evaluation is essential for improvement
- Corrective actions are essential
