# Video Script: Validation - Unit 4.1
## Analytical Method Validation
**Duration:** 22 minutes

---

### Introduction (0:00 - 1:00)
**Speaker:** In this lecture, we will discuss Analytical Method Validation. Analytical Method Validation is essential to ensure that analytical methods are accurate and reliable.

### What is Analytical Method Validation? (1:00 - 5:00)
**Speaker:** Analytical Method Validation is the process of demonstrating that an analytical method is suitable for the intended purpose.

**Definition:**
- Demonstrating that the method is suitable for the purpose
- Demonstrating that the method is accurate
- Demonstrating that the method is reliable
- Demonstrating that the method is consistent

**Importance of Analytical Method Validation:**
- Ensures result accuracy
- Ensures result reliability
- Ensures result consistency
- Essential for regulatory compliance

**Detailed Explanation:**
- Analytical Method Validation is required by ICH Q2(R2)
- Analytical Method Validation must be performed before use
- Analytical Method Validation must be documented

### ICH Q2(R2) Criteria (5:00 - 12:00)
**Speaker:** ICH Q2(R2) defines specific criteria for Analytical Method Validation.

**Main Criteria:**

**1. Specificity:**
- Ability to accurately measure target substance
- Ability to distinguish between different substances
- Ability to detect interferences
- Ability to detect impurities

**2. Linearity:**
- Ability to produce results proportional to concentration
- Required linear range
- Correlation coefficient
- Coefficient of determination

**3. Accuracy:**
- Closeness to true value
- Comparison with approved reference
- Recovery studies
- Comparison with reference method

**4. Precision:**
- Repeatability
- Intermediate Precision
- Reproducibility
- Standard Deviation

**5. Detection Limit:**
- LOD (Limit of Detection)
- Lowest concentration that can be detected
- Signal-to-Noise ratio
- Standard Deviation of blank

**6. Quantitation Limit:**
- LOQ (Limit of Quantitation)
- Lowest concentration that can be measured accurately
- Signal-to-Noise ratio
- Standard Deviation

**7. Robustness:**
- Ability to withstand small changes
- Changes in parameters
- Changes in conditions
- Changes in equipment

### Analytical Method Validation Procedures (12:00 - 18:00)
**Speaker:** Executing Analytical Method Validation requires following specific procedures.

**Steps:**

**1. Method Development:**
- Develop analytical method
- Determine optimal conditions
- Define parameters
- Define procedures

**2. Protocol Development:**
- Develop validation protocol
- Define required criteria
- Define acceptance criteria
- Define procedures

**3. Validation Execution:**
- Execute required tests
- Collect data
- Analyze results
- Address any problems

**4. Results Analysis:**
- Analyze data
- Compare results with criteria
- Identify any deviations
- Identify any problems

**5. Documentation:**
- Document all activities
- Prepare validation report
- Review report
- Approve report

### Analytical Method Validation Example (18:00 - 22:00)
**Speaker:** Let's look at a practical example of Analytical Method Validation.

**Example: Validation of HPLC Method:**

**Specificity:**
- Test distinction between target substance and impurities
- Test interference detection
- Test degradation product detection

**Linearity:**
- Prepare 5-7 different concentrations
- Measure response for each concentration
- Calculate correlation coefficient
- Calculate coefficient of determination

**Accuracy:**
- Add known amounts of target substance
- Measure Recovery
- Compare with approved reference

**Precision:**
- Execute 6 repeated tests
- Calculate Standard Deviation
- Calculate Relative Standard Deviation

**Detection Limit:**
- Measure Signal-to-Noise ratio
- Calculate LOD based on 3:1 S/N
- Calculate LOQ based on 10:1 S/N

**Robustness:**
- Change temperature ±2°C
- Change solvent flow ±10%
- Change solution pH ±0.2
- Evaluate impact on results
