### Root Cause Analysis Tools (6:00 - 10:00)
**Speaker:** Root cause analysis tools are essential to determine the root cause.

**Main Tools:**

**1. 5 Whys:**
- Simple and fast
- Effective for simple problems
- Requires analysis skill
- May not be sufficient for complex problems

**2. Fishbone Diagram:**
- Comprehensive and organized
- Effective for complex problems
- Provides comprehensive view
- Requires longer time

**3. Pareto Analysis:**
- Helps determine priorities
- Effective for recurring problems
- Focuses on main causes
- Requires historical data

**4. Fault Tree Analysis:**
- Logical analysis
- Effective for technical problems
- Provides deep analysis
- Requires technical expertise

### Problem Management Tools (10:00 - 14:00)
**Speaker:** Problem management tools are essential to track and manage all problems.

**Main Tools:**

**1. Problem Tracking System:**
- Track all problems

- Track responsibilities
- Track timelines

**2. Problem Database:**
- Store all problems
- Analyze historical data
- Identify trends
- Identify recurring problems

**3. Complaint Management System:**
- Manage customer complaints
- Manage patient complaints
- Manage doctor complaints
- Manage pharmacist complaints

**4. Deviation Management System:**
- Manage process deviations
- Manage quality deviations
- Manage specification deviations
- Manage regulatory deviations
