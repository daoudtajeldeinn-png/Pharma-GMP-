# Video Script: CAPA - Unit 1.3
## CAPA Process
**Duration:** 20 minutes

---

### Introduction (0:00 - 1:00)
**Speaker:** In this lecture, we will discuss the CAPA process. Understanding the process is essential to effectively implement CAPA.

### CAPA Process Stages (1:00 - 8:00)
**Speaker:** The CAPA process includes several organized stages.

**Main Stages:**

**1. Problem Identification:**
- Identify the problem
- Describe the problem
- Determine problem impact
- Determine problem priority

**2. Investigation:**
- Collect information
- Analyze data
- Determine root cause
- Document results

**3. Action Development:**
- Develop corrective actions
- Develop preventive actions
- Define timeline
- Define responsibilities

**4. Action Implementation:**
- Execute actions
- Monitor implementation
- Address problems
- Document

**5. Effectiveness Verification:**
- Verify action effectiveness
- Measure results
- Analyze data
- Evaluate effectiveness

**6. Review and Closure:**
- Review results
- Close problem
- Update records
- Document

### Problem Identification (8:00 - 12:00)
**Speaker:** Problem identification is the first and most important step in the CAPA process.

**Procedures:**

**1. Collect Information:**
- Collect information about the problem
- Collect information about impact
- Collect information about context
- Collect information about history

**2. Describe Problem:**
- Describe problem clearly
- Describe problem impact
- Describe problem timing
- Describe problem location

**3. Determine Priority:**
- Determine problem severity
- Determine problem impact
- Determine problem frequency
- Determine priority

**4. Report:**
- Report the problem
- Report to supervisor
- Report to management
- Report to concerned parties

### Problem Investigation (12:00 - 16:00)
**Speaker:** Problem investigation is essential to determine the root cause.

**Procedures:**

**1. Collect Evidence:**
- Collect data
- Collect records
- Collect testimonies
- Collect information

**2. Analyze Data:**
- Analyze collected data
- Analyze trends
- Analyze patterns
- Analyze relationships

**3. Determine Root Cause:**
- Use root cause analysis tools
- Determine true root cause
- Verify root cause
- Document root cause

**4. Document Results:**
- Document all activities
- Document all results
- Document all decisions
- Document all actions

### Action Development (16:00 - 20:00)
**Speaker:** Developing corrective and preventive actions is essential to solve the problem and prevent recurrence.

**Procedures:**

**1. Develop Corrective Actions:**
- Develop actions to correct current problem
- Define timeline
- Define responsibilities
- Define resources

**2. Develop Preventive Actions:**
- Develop actions to prevent future problems
- Define timeline
- Define responsibilities
- Define resources

**3. Approve Actions:**
- Review actions
- Approve actions
- Document
- Sign

### Summary (20:00)
**Speaker:** In this lecture, we learned:
- CAPA process stages
- Problem identification
- Problem investigation
- Action development

**Key Points to Remember:**
- CAPA process includes several stages
- Each stage is essential
- Documentation is essential in each stage
- Root cause is essential for effective solution

In the next lecture, we will discuss Root Cause Analysis.
