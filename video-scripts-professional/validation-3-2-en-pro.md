Video Script: Validation - Unit 2 OQ (Operational Qualification) Duration: 20 minutes (0:00 - 1:00) In this lecture, we will discuss OQ or Operational Qualification.

OQ is the second stage of validation and is essential to ensure that equipment operates correctly.

What is OQ?

(1:00 - 5:00) OQ is the verification of equipment operation.

Operational qualification Ensure equipment operates according to specifications Ensure all functions operate Ensure equipment is safe to operate of OQ: Ensures correct operation Reduces operational errors Ensures measurement accuracy Essential before PQ OQ must be performed after IQ OQ must be fully documented OQ is essential before PQ OQ Contents (5:00 - 12:00) OQ includes several elements that must be verified.

Main Elements: Functional Testing: Test all functions Test all modes Test all speeds Test all parameters Accuracy Testing: Test measurement accuracy Test control accuracy Test time accuracy Test temperature accuracy Repeatability Testing: Test result repeatability Test result consistency Test result stability Test result reliability Safety Testing: Test safety systems Test emergency stop Test overheat protection Test overpressure protection Limit Testing: Test upper limits Test lower limits Test critical limits Test stability at limits OQ Procedures (12:00 - 17:00) Executing OQ requires following specific procedures.

Steps: Review Protocol: Review OQ protocol Review acceptance criteria Review test procedures Review required resources Equipment Preparation: Ensure IQ completion Ensure equipment readiness Ensure material availability Ensure employee training Execute Tests: Execute functional tests Execute accuracy tests Execute repeatability tests Execute safety tests Data Collection: Record all data Record all observations Record any deviations Record any problems Result Analysis: Analyze data Compare results with criteria Identify any deviations Identify any problems Documentation: Document all activities Prepare OQ report Review report Approve report OQ Example (17:00 - 20:00) Let's look at a practical example of OQ.
