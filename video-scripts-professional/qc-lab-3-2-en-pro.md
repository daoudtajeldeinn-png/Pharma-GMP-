What are External Tests?

(1:00 - 5:00) External tests are tests where laboratory results are compared with results from other laboratories.

Comparison tests between laboratories Proficiency tests Calibration tests Comparison tests of External Tests: Ensures laboratory competence Ensures result accuracy Ensures result reliability Essential for accreditation External tests must be periodic External tests must be comprehensive External tests must be documented External tests must be followed up Types of External Tests (5:00 - 10:00) There are several types of external tests.
